# Cambios incluidos en la versión 2

- Galería construida con JavaScript y `forEach()`.
- Fechas, títulos, descripciones, categorías y filtros.
- Vista a pantalla completa con navegación.
- Botón al álbum público de Google Fotos.
- Generador Python para optimizar fotos y crear el JSON.
- Lectura de fechas desde Google Takeout y EXIF.
- Miniaturas para acelerar GitHub Pages.
- Reproductor de playlist corregido con `listType=playlist`.
- Lista visible opcional de todas las canciones.
- Integración automática con YouTube Data API y paginación.
- Alternativa sin clave mediante `assets/data/songs.json`.
